How to read a time signature

A time signature appears as two numbers stacked on top of each other, like a fraction — but it isn't one.

4 4
← how many beats are in one measure
← what note value gets one beat
Top number
Beats per measure

How many beats you count before the pattern repeats. In 4/4 you count 1 2 3 4, in 3/4 you count 1 2 3.

Bottom number
The beat unit

Which note value equals one beat. 4 means a quarter note, 8 means an eighth note, 2 means a half note.

2 = half note 4 = quarter note 8 = eighth note 16 = sixteenth note
