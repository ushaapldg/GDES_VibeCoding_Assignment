Reading Rhythms

Rhythm is how long each note lasts. Every note type has a beat value — the number of beats you hold it for. Learning to count out loud is the fastest way to lock in solid rhythm.

The whole rest hangs down from a line (heavy block, sinks down); the half rest sits up on a line (like a hat). Every rest takes up the same number of beats as the matching note — keep counting through the silence.
